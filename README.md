# Bank AI Loan & Bias Detector 🏦

This project is a web-based application designed to predict loan approval while using Generative AI (Gemini) to explain the decision and detect potential biases in the model.

##  Features
- **Machine Learning**: Random Forest model to predict loan risk.
- **Explainable AI (XAI)**: Uses Google Gemini API to provide fairness reports.
- **Bias Detection**: Analyzes if factors like Age or Gender influenced the decision.
- **User Interface**: Simple web dashboard for bank managers.

##  Technology Stack
- **Backend**: Python, Flask
- **AI/ML**: Scikit-Learn, Joblib, Pandas
- **Generative AI**: Google Gemini API (`google-genai`)
- **Frontend**: HTML5, CSS3, JavaScript
